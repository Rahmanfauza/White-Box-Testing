*Control Flow Testing* adalah teknik yang menguji semua kemungkinan jalur eksekusi dalam suatu program berdasarkan diagram alur kontrol (*Control Flow Graph*). Pengujian ini bertujuan untuk menjamin bahwa semua percabangan, kondisi, dan loop dalam program telah diuji secara menyeluruh. Teknik ini sangat berguna untuk mendeteksi kesalahan logika yang terjadi dalam struktur kendali program.
